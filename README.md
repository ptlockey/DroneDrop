# Drone Drop Game

A physics-based drone drop simulation game built with Streamlit and Folium.

## Features
- Simulates realistic drone drop trajectories with wind, drag, and environmental effects
- Interactive map and controls
- Randomized cities and land points for each round

## Setup
1. Clone the repository.
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the app:
   ```
   streamlit run main.py
   ```

## Project Structure
- `main.py` — Entry point for the Streamlit app
- `drone_drop/` — Game logic and utilities
- `requirements.txt` — Python dependencies
